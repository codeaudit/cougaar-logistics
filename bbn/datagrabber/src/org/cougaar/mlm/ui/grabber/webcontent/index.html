The Data Gatherer

An integrated environment for COUGAAR aggregation and assessment

LIST AVAILABLE RUNS View current and previously available runs done by the data gatherer.
COMPLETION ASSESSOR View the completion status of the current COUGAAR society, if one is running.
LIST ACTIVE WORK List active work that is currently being done by the data gatherer.
LIST COMPLETE LOGS List data gatherer logs for each previous run.
VALIDATION View and run validation tests that assess the correctness of previous society runs.
COMPARISON View and run comparisons between validation tests of different runs.
DERIVED TABLES Generate derived tables that enhance TPFDD performance.
